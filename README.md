# Blockchain Solutions Test — Messenger

## Установка
* Создать виртуальное окружение
* Установить пакеты из `requirements.txt`
* Создать `.env` и заполнить переменные окружения c помощью `python-dotenv` (BS_DJANGO_MESSENGER_SECRET_KEY, BS_DJANGO_MESSENGER_SERVER_IP, BS_DJANGO_MESSENGER_DB_NAME, BS_DJANGO_MESSENGER_DB_USER, BS_DJANGO_MESSENGER_DB_PASS, BS_DJANGO_MESSENGER_DB_HOST, BS_DJANGO_MESSENGER_DB_PORT)
* Выполнить миграции 

## Техническое задание
Разработать веб-версию чата со следующим функционалом(Обязательный стек: Python + Django + PostgreSQL):
* a. Форма регистрации/авторизации (проверка на занятость никнейма)
* b. Поиск собеседника по никнейму  
* i. В случае обнаружения собеседника, появляется возможность начать с ним
приватную переписку
* ii. В случае отсутствия никнейма в БД, выводим ошибку
* c. Хранение истории переписки
* d. Возможность блокировать пользователей из Django
* Фронтовая часть должна иметь чисто технический вид (не акцентируйте внимание на
визуальной части)
* Любые задачи, которые не указаны в описании тестового задания и на ваш взгляд
требующие их выполнения, решайте так как считаете нужным
